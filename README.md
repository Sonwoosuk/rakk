## Project Overview

락고재의 브랜드 이미지는 유지하면서 정보 구조와 사용자 경험을 개선하는 것을 목표로 진행한 프로젝트입니다.

기획부터 디자인, 퍼블리싱, Firebase 연동, 배포까지 전 과정을 직접 진행했습니다.

## Main Features

- Google 로그인
- 반응형 웹 지원
- 객실 정보 제공
- 예약 확인 페이지
- 카카오 지도 연동
- 로딩 애니메이션
- 마이페이지 UI
